## 6.4 定制消息模版
  报错模版: 
  ```
  抱歉出错啦，错误代码：${error}，你可以访问 https://aow.me/faq 了解更多信息
  ```
  超过提醒次数触发模版: 
  ```
  抱歉，你的提问太频繁了，你可以访问 https://aow.me 了解更多信息
  ```
  群组触发模版: 
  ```
  抱歉，该群组已触发当日对话上限。你可以访问 https://aow.me 了解更多信息
  ```
> 如果您需要定制消息模版，请在购买MAX版本后将您要定制的消息模版，发送邮件至support@aow.me
