我们提供了详细的指南和技术支持，帮助您在微信中使用ChatGPT For WeChat插件。以下是在WeChat中使用插件的功能介绍：

## 4.1 支持 ChatGPT 网页端 
选择使用 ChatGPT 网页端，您可以随时在微信中获得与在ChatGPT网页一样的互动体验效果，获取实时的聊天回复。
### 使用方法
    - 打开 https://chat.openai.com
    - 保持登录状态
### 使用过程可能存在的问题
- 无响应，请确保您的浏览器版本为最新版本
- 请确保您已经登录了自己的浏览器账号
- 非ChatGPT Plus账号用户，建议优先使用API。
- ChatGPT Plus 账号使用时，可以选择 gpt4的模型。但是次数有限，如果不响应或者出错，请更换模型再试。

## 4.2 ChatGPT API

API的模型可选ChatGPT 3.5和gpt4.0。
选择模型后，您需要填写自己的API-Key。获取API-Key的地址：
以下是获取API密钥和使用API接口的步骤：

- 登录网站，进入“API管理”页面，https://platform.openai.com/account/api-keys
- 点击“生成新密钥”按钮，系统将为您生成一个API密钥。
- 将API-Key复制到插件内，保存后使用。
请注意，gpt4.0的模型需要申请获得权限后才可以使用，请自行前往申请：https://openai.com/waitlist/gpt-4-api

##### 可能出现的问题
如果您在使用API的过程，无响应。您需要确保您的key有效，且在有效期内。
- 前往 https://status.openai.com 查看openai的api的状态。
- 前往 https://platform.openai.com/playground 查看key是否有效
- 使用过程，出现其他报错问题，请查看[错误列表](./error_codes_and_solutions)
- 选择gpt4.0模型前，你需要在openai官网申请权限。无权限不可使用gpt4.0

## 4.3 一键开启 Bot 接管

通过一键开启 Bot 接管功能，您可以轻松地让AI接管您的聊天，为您的好友提供智能回复和帮助。
- 开启 Bot 接管，表示您的微信同意接受ChatGPT返回的信息，即变为机器人的角色。
- 关闭 Bot 接管，您的微信则不再接受ChatGPT的信息。

## 4.4 入群欢迎语

当有新成员加入群聊时，AI可以自动发送欢迎语，帮助新成员更好地融入群聊。

```
欢迎 「新用户昵称」 加入本群！我是「登录微信昵称」，
进群后请先阅读下本群公告哦，也可以@我回答你任何问题。
```
