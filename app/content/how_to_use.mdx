## 2.1 安装插件
要开始使用ChatGPT For WeChat插件，请先完成插件的安装：

### 2.1.1 Chrome 市场安装方法

1. 前往 [Chrome市场](https://chrome.google.com/webstore/detail/chatgpt-for-wechat/cdlangplaophialnpfbdfndiobanklfd)
2. 在添加插件到扩展程序，点击 【add to Chrome】安装。
3. 安装成后，点亮插件小图钉。然后点开插件，看到插件主页面 ChatGPT For WeChat。

安装完成后，接下来您可以开始设置插件。

### 2.1.2 离线包安装
```
1. 请前往官网https://aow.me 下载离线包，最新版本为 4.1.6
2. 下载完成后，请将离线包，拖入扩展程序空白处自动安装
3. 扩展程序地址：chrome://extensions/
```
![Chrome 扩展程序地址](/kz.png)

`注意事项`：
1. 推荐使用最新版本的Chrome浏览器，您可以选择使用 Edge、Firefox。
2. 如果安装失败，请登录Chrome 浏览器账号，并查看开发者模式是否打开状态。
![Chrome开发者模式](/kf.png)

## 2.2 配置插件
在使用插件之前，请先进行配置，打开插件，点击“设置”菜单,点击进入。
![设置菜单](/set.png)
### 2.1.1 AI源配置
```
1. 非ChatGPT Plus账号用户，建议优先使用API。
2. 选择模型时，请注意，API没有gpt4.0权限，不要选gpt4.0。
3. 连接测试时，一定要先保存配置，再连接测试！
```
![选择AI源](/step1.png)

### 2.1.2 设置bot
```
- 一键开启 Bot 项必须开启。否则无响应！如果不想接受AI回应，就关闭。
- 新用户入群欢迎，在调试时，可以开启。开启后，拍一拍，如果有回应，表示已与微信连接成功！
```
![bot设置项](/step2.png)

### 2.1.3 设置完成后，点击插件图标，选择AI源
```
⚠️ 请注意，设置完成后，一定要点击插件图标，选择要使用的AI源，出现连接正常，就可以正常使用。
```
![选择AI源](/step00.png)

## 2.3 与 ChatGPT 互动
使用插件与 ChatGPT 进行互动非常简单，在客户端微信（手机微信或桌面端微信）中随意打开一个的聊天窗口，群聊或私聊都可以。

### 2.3.1 群聊用法
```
群聊聊天窗口中输入：@登录微信昵称 你好
```
ChatGPT 将根据您的输入生成回复，并显示在聊天窗口中。`注意：在网页微信窗口内@是无效的。`

### 2.3.2 私聊用法

1. 非Max版本用户：`@微信昵称 问题`；
2. Max用户可以调整参数，改为总是触发后，就可以直接私聊！

### 2.3.3 使用文件传输助手，作为自己的个人智能助手
1. 担心微信封号的小伙伴，可以选择文件传输助手，这个方式，至今没有反馈被警告或封号。
2. 点击前往并登录 https://szfilehelper.weixin.qq.com/
3. 配置同上，使用命令
```
@gpt 讲个笑话
```
### 2.3.3 其他命令
1. 重置对话

```
@微信昵称 重置对话
```
清空上下文，重新开启新的对话。

2. 拍一拍

开启新用户进群，便能开启拍一拍。方便用户了解机器人是否在线。

### 2.3.4 注意事项：
- 在网页微信中提问是无效的，需要在客户端微信提问，手机微信或桌面微信中。
- 提问的话题或问题要大于2个字。

您可以随时与 ChatGPT 进行对话，获取有趣且实用的信息。

### 2.3.5 使用技巧
- 在企业微信外部群中使用
- 建议微信名不要使用：人工智能、gpt、ChatGPT、ai、bot、chatbot、机器人等。
- 微信网页登录不要长时间在线。或者不定时更换不同微信号
- Pro和Max版本的用户，在提供服务时，可以在prompt中限定AI回复字数，以及限定群组提问次数、群组用户提问次数，来保证账号的安全。
- 如果社群人数较多，消息频率较高，建议通过浏览器多账户登录不同的微信号来提供服务。

## 2.4 更新与卸载插件
我们会定期发布插件更新，以修复问题和增加新功能。请定期检查更新，以确保您使用的是最新版本的插件。

如需卸载插件，请按照以下步骤操作：

1. 打开您的扩展程序管理器。
2. 找到“ChatGPT For WeChat插件”选项，点击移除。
3. 按照提示进行操作。

卸载插件后，您将无法继续使用插件提供的功能。如有需要，您可以重新下载并安装插件。

希望本使用指南能帮助您顺利使用ChatGPT For WeChat插件。如在使用过程中遇到无法响应的问题，请查看[常见问题](./frequently_asked_questions)。
