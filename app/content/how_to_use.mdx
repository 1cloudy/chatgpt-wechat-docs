## 2.1 安装插件
要开始使用ChatGPT For WeChat插件，请先完成插件的安装：

```
请前往官网https://aow.me 下载离线包，最新版本为 4.1.6
```

1. 前往 [Chrome市场](https://chrome.google.com/webstore/detail/chatgpt-for-wechat/cdlangplaophialnpfbdfndiobanklfd)
2. 在添加插件到扩展程序，点击 【add to Chrome】安装。
3. 安装成后，点亮插件小图钉。然后点开插件，看到插件主页面 ChatGPT For WeChat。

安装完成后，接下来您可以开始设置插件。
备注：您也可以根据自己使用习惯，选择在edge、firefox等浏览器中安装使用。

## 2.3 与 ChatGPT 互动
使用插件与 ChatGPT 进行互动非常简单，在客户端微信（手机微信或桌面端微信）中随意打开一个的聊天窗口，群聊或私聊都可以。

### 2.3.1 群聊用法
```
群聊聊天窗口中输入：@登录微信昵称 你好
```
ChatGPT 将根据您的输入生成回复，并显示在聊天窗口中。`注意：在网页微信窗口内@是无效的。`

### 2.3.2 私聊用法
```
非Max版本用户：@微信昵称 问题；Max用户可以调整参数，改为总是触发后，就可以直接私聊！
```

3. 需要清空上下文，重新开启新的对话，请发送 `重置对话` 或 `重置` 的命令

### 注意事项：
- 在网页微信中提问是无效的，需要在客户端微信提问，手机微信或桌面微信中。
- 提问的话题或问题要大于2个字。

您可以随时与 ChatGPT 进行对话，获取有趣且实用的信息。

## 2.4 更新与卸载插件
我们会定期发布插件更新，以修复问题和增加新功能。请定期检查更新，以确保您使用的是最新版本的插件。

如需卸载插件，请按照以下步骤操作：

1. 打开您的扩展程序管理器。
2. 找到“ChatGPT For WeChat插件”选项，点击移除。
3. 按照提示进行操作。

卸载插件后，您将无法继续使用插件提供的功能。如有需要，您可以重新下载并安装插件。
